# marvel-catalog
Application will be made in order to study new libraries, it will use the official api of marvel.

https://developer.marvel.com/


### Libraries

- Hilt
- Ktor
- Room
- Android Navigation
- Kotlin Flow
- View Binding
- Firebase

### Features

- Characters List
- Characters Details
- Comics List
- Comics Details
- Creators List
- Creators Details

### Technical Characteristics

- Clean Architecture
- Modularization
- Grandle Flavors
- Dependency versions file
- Tested data and domain modules
- CI
